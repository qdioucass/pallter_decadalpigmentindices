SUPPORTING CODE & DATA FOR MANUSCRIPT "Decadal increase in phytoplankton community photoprotective pigments associated with shoaling mixed layers along a warming West Antarctic Peninsula"

Diou-Cass, Q.P., Waite, N. and Schofield, O. (2026), Decadal increase in phytoplankton community photoprotective pigments associated with shoaling mixed layers along a warming West Antarctic Peninsula. Limnol Oceanogr, 71: e70324. https://doi.org/10.1002/lno.70324

The "data_code_analysis" folder in this repository contains all necessary initial data and code to run the analysis performed for the associated study.
Running the associated code requires Python (v3.11 or later).
Download this repository and follow the instructions in "README_RunningAnalysis.docx" in the "data_code_analysis" folder.
